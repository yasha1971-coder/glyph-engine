Evidence chain, not search results.

Search tools return matches. GLYPH’s current research target is different: make exact-byte retrieval results replayable by another person from a fixed corpus.

Where GLYPH sits.

GLYPH should not be compared as a faster grep, a SIEM, or a malware scanner. The relevant question is whether a system produces a portable, replayable exact-byte evidence artifact.

System class Main job Exact bytes Replay artifact Portable bundle GLYPH relation
grep / ripgrep Fast local text search partial no no GLYPH is not replacing grep; it records replayable evidence.
Elastic / Splunk Search, observability, security analytics contextual no no GLYPH is not a platform; it is a byte-evidence layer.
YARA Textual/binary pattern matching for malware research yes no no Future possible fit: evidence artifact for confirmed pattern hits.
OpenTimestamps / CT / Sigstore Time, transparency, signing, provenance no search different layer different layer Complementary trust layer, not exact retrieval.
GLYPH current checkpoint Replayable exact-byte bounded evidence yes yes yes Experimental reference path over fixed sentinel-safe corpora.

Boundaries are part of the design.

GLYPH’s current public checkpoint is deliberately narrow. The goal is reviewable engineering, not inflated claims.

Current strongest claim: GLYPH has a reproducible review checkpoint for replayable exact-byte bounded evidence over fixed sentinel-safe corpora.

Not binary-safe yet

0x00 support is documented as a boundary. Reference semantics exist, but production C++ runtime support is not claimed.

Not a legal proof

No timestamp, signature, notarization, or chain-of-custody claim is made by the current checkpoint.

Not distributed

The current verified path is a single-repository review fixture, not petabyte-scale distributed infrastructure.

24-hour reviewer portal.

GLYPH’s public page is designed to work without a meeting. A reviewer should be able to open the repository, inspect the technical state, run the checkpoint, and see exactly what is verified and what is not claimed.
