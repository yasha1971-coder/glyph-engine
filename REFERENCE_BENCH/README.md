# REFERENCE_BENCH

Purpose:

Study GLYPH against known reference corpora and systems.

References:

1. Pizza & Chili Corpus
2. SDSL-lite
3. ripgrep

Goal:

Not to claim victory.

Goal is to understand:

- corpus behavior
- index size ratio
- build time
- query latency
- memory footprint
- architecture lessons

Current rule:

Download small canonical corpora first.

Do not start with huge datasets.
Do not modify GLYPH core.
Do not mix reference artifacts with main benchmark artifacts.
