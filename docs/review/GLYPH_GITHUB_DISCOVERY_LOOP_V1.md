# GLYPH_GITHUB_DISCOVERY_LOOP_V1

Status: active discovery loop  
Date: 2026-06-28

## Purpose

Create a low-friction GitHub path for people who cloned, starred, forked, or ran GLYPH to tell us why they looked.

This is not telemetry.

This is a voluntary feedback loop.

## Added

- `.github/ISSUE_TEMPLATE/use_case_feedback.yml`
- `.github/ISSUE_TEMPLATE/config.yml`
- README section: `Who is this for?`
- `verify.sh` post-success feedback link

## Feedback target

https://github.com/yasha1971-coder/glyph-engine/issues/new/choose

## Goal

Find real cases where replayable exact-byte evidence over fixed corpora matters.

## Non-goals

- Do not spam stargazers.
- Do not claim adoption from stars.
- Do not confuse clones with users.
- Do not treat polite comments as pain.
- Do not build major features before real cases are found.

## Fail criterion

If people run or star GLYPH but nobody can describe a real case where replayable exact-byte evidence matters, the project should not assume product relevance from technical capability alone.
