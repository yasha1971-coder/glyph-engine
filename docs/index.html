GLYPH

Persistent byte-level indexed retrieval over static corpora.

Benchmark

Dataset: HDFS 1GB
Queries: 100 exact blk_* lookups

grep repeated scan: 11.516 sec
GLYPH persistent FM: 0.001673 sec total
Average: 0.0167 ms/query

Cost:
Persistent server RAM: ~9.4GB
Index artifacts: ~13.1GB

Tradeoff

GLYPH trades RAM and offline indexing for extremely fast repeated exact queries.
